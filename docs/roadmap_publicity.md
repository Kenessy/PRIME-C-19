# Publicity Timeline (Trigger-Based)

This is a simple “when X, do Y” roadmap. We only announce after a milestone is
repeatable (>= 3 seeds unless noted).

## Milestone 1 — Seam Fix Proved

**X:** Ring wrap is stable and reproducible (no teleport, no NaNs, 3 seeds).  
**Y:** Small Reddit update + short blog note (1 chart, 1 GIF, 5 bullets).

## Milestone 2 — Small Bench Win

**X:** Spiral + sine regression hit stable targets across 3-5 seeds
(e.g., spiral acc >= 0.95, sine MSE <= 0.01).  
**Y:** “Evidence drop” post + short notebook/Colab demo (< 5 minutes to run).

## Milestone 3 — Long-Range Recall

**X:** assoc_clean reaches >= 0.80 accuracy on >= 3 configs, ablation shows
the fix matters (cadence + fractional kernel).  
**Y:** arXiv/OpenReview short technical note + targeted outreach to memory-model
folks.

## Milestone 4 — Baseline Beat

**X:** Beats a standard baseline on a comparable budget (seq-MNIST or LRA).  
**Y:** Broad announcement (HN/Twitter/YouTube short).

## Milestone 5 — External Repro

**X:** Independent reproduction confirmed.  
**Y:** “Confirmed” post + full paper draft.
